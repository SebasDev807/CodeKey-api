# CodeKey API
## Este Proyecto utiliza yarn como gestor de dependencias
Si no tienes yarn instalalo globalmente
```
npm install - g yarn
```

1. Asegurate de tener Nest CLI instalado
```
npm install -g @nestjs/cli
```
